# Brand assets

| File | Use |
|---|---|
| `logo.svg` | Source vector — the "verified human" mark on a coral squircle |
| `logo.png` (512²) / `logo-1024.png` (1024²) | Raster avatar — **org avatar** (covers the repo avatar too) |
| `social-preview.png` (1280×640) | **Repo → Settings → Social preview** banner |

Palette: coral `#E07A5F`, slate `#3D405B`, white. Rendered from `logo.svg`
with ImageMagick (`magick -background none logo.svg -resize 512x512 logo.png`).
